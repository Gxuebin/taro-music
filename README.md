# taro-music

## Introduction

 基于 [taro](https://github.com/NervJS/taro) 开发的音乐播放器的小程序，后台是网易云音乐  [NeteaseCloudMusicApi](https://binaryify.github.io/NeteaseCloudMusicApi/#/) 提供的api

## Usage

> 在运行项目之前需要先在本地运行 NeteaseCloudMusicApi 项目，开启获取歌曲服务

```bash
# install
$ npm i
# or yarn install
$ yarn install
# development
$ npm run dev:weapp
# production
$ npm run build:weapp
```
